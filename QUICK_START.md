# 🚀 智能开单应用 - 快速启动指南

## 📱 立即体验

### 1. 访问应用
在浏览器中打开: **http://localhost:5174/**

### 2. 快速导航
- **控制台** (`/`) - 查看销售概览和订单统计
- **智能开单** (`/billing`) - 创建新订单，支持AI图片识别
- **客户管理** (`/customers`) - 管理客户信息
- **需求分析** (`/analysis`) - 查看销售数据分析
- **系统设置** (`/settings`) - 配置API密钥和系统偏好

## 🎯 核心功能演示

### 📊 1. 查看销售概览
1. 访问首页控制台
2. 查看今日销售、总销售额等统计数据
3. 点击眼睛图标可隐藏/显示金额
4. 切换"赊欠订单"和"全部订单"标签

### 🧾 2. 创建订单 (手动)
1. 进入"智能开单"页面
2. 点击"添加商品"按钮
3. 输入商品名称、数量、价格
4. 选择客户 (可选)
5. 点击"提交订单"

### 🤖 3. AI智能识别 (需要API Key)
1. 先到"系统设置"页面配置 Gemini API Key
2. 回到"智能开单"页面
3. 点击"上传图片"按钮
4. 选择包含商品信息的图片
5. AI会自动识别并填充商品信息

### 👥 4. 管理客户
1. 进入"客户管理"页面
2. 查看现有客户列表
3. 点击"添加客户"创建新客户
4. 使用搜索框快速查找客户

## 🔧 配置说明

### API Key 设置
1. 访问 [Google AI Studio](https://makersuite.google.com/app/apikey)
2. 创建新的 API Key
3. 在应用的"系统设置"页面粘贴 API Key
4. 保存设置

### 数据存储
- 所有数据保存在浏览器本地存储中
- 清除浏览器数据会丢失所有信息
- 支持数据导入/导出功能 (在设置页面)

## 🔍 故障排除

### 页面无法加载
```bash
# 重启开发服务器
npm run dev
```

### 功能异常
1. 刷新浏览器页面
2. 检查浏览器控制台错误
3. 清除浏览器缓存

### AI功能不工作
1. 确认API Key已正确配置
2. 检查网络连接
3. 验证图片格式 (支持 JPG, PNG, WebP)
