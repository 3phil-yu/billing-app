## 🐛 常见问题排查

### 如果页面无法加载:
1. 检查开发服务器是否运行: `npm run dev`
2. 检查端口是否被占用
3. 清除浏览器缓存

### 如果 AI 功能不工作:
1. 确认已在设置页面配置 Gemini API Key
2. 检查网络连接
3. 查看浏览器控制台错误信息

### 如果数据丢失:
1. 检查浏览器本地存储
2. 确认没有清除浏览器数据
3. 重新添加测试数据
